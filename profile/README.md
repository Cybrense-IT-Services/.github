### CBS Mail

**CBS Mail** is our open-source Roundcube webmail experience: a branded,
mobile-ready interface and plugin layer with PWA support, Docker deployment,
per-user labels, and a privacy-aware remote-content flow.

## How we work

- **Security by design** - threat-aware decisions are part of engineering, not a final checklist.
- **Built for ownership** - clear documentation, maintainable code, and reproducible delivery.
- **Quality in the workflow** - automation and testing provide fast, useful feedback.
- **Open collaboration** - focused issues, reviewable changes, and respectful technical discussion.
